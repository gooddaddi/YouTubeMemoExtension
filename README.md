# YouTube Memo Extension (유튜브 메모장)

유튜브 시청 중 다시 보고 싶은 영상을 카테고리별로 분류하고 간단한 메모와 함께 저장할 수 있는 크롬 확장 프로그램입니다.

## 📌 주요 기능
- **자동 정보 수집**: 현재 보고 있는 유튜브 영상의 제목과 URL을 자동으로 가져옵니다.
- **나만의 카테고리**: '공부', '음악', '재미' 등 원하는 카테고리를 직접 추가하고 괸리할 수 있습니다.
- **메모 작성**: 영상에 대한 간단한 노트나 타임스탬프 등을 기록할 수 있습니다.
- **리스트 관리**: 저장된 영상들을 리스트 형태로 한눈에 확인할 수 있습니다.
- **정렬 및 필터링**:
    - 카테고리별로 모아보기
    - 최신순 / 오래된순 / 카테고리순 정렬 저원
- **눈이 편안한 디자인**: 다크 네이비와 소프트 블루 컬러를 사용하여 눈의 피로를 줄인 모던한 UI를 제공합니다.

## 🚀 설치 방법

이 확장 프로그램은 현재 크롬 웹 스토어에 등록되어 있지 않으며, **개발자 모드**를 통해 로컬에서 설치하여 사용할 수 있습니다.

### 1. 소스 코드 다운로드
- 이 저장소를 `git clone` 하거나 ZIP 파일로 다운로드하여 압축을 풉니다.
  ```bash
  git clone https://github.com/YOUR_USERNAME/YouTubeMemoExtension.git
  ```

### 2. 크롬 확장 프로그램 로드
1. Chrome 브라우저를 실행하고 주소창에 `chrome://extensions`를 입력합니다.
2. 우측 상단의 **개발자 모드 (Developer mode)** 스위치를 켭니다.
3. 좌측 상단의 **압축해제된 확장 프로그램을 로드합니다 (Load unpacked)** 버튼을 클릭합니다.
4. 다운로드 받은 폴더(`YouTubeMemoExtension`)를 선택합니다.

### 3. 사용하기
- 유튜브 영상 페이지로 이동합니다.
- 브라우저 우측 상단의 확장 프로그램 아이콘(퍼즐 조각 모양)을 클릭하여 **YouTube Memo**를 고정(핀)하면 더 편하게 쓸 수 있습니다.
- 아이콘을 클릭하여 메모를 작성하고 저장하세요!

## 🛠 기술 스택
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Storage**: Chrome Storage API (`chrome.storage.local`)
- **Manifest**: Manifest V3
