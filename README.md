# YouTube Memo Extension (유튜브 메모장)

유튜브 시청 중 다시 보고 싶은 영상을 카테고리별로 분류하고 간단한 메모와 함께 저장할 수 있는 크롬 확장 프로그램입니다. 최신 업데이트를 통해 더욱 고급스러운 **다크 모드**와 **영상 썸네일 배경** 기능을 제공합니다.

## 📌 주요 기능
- **자동 정보 수집**: 현재 보고 있는 유튜브 영상의 제목과 URL을 자동으로 가져옵니다.
- **나만의 카테고리**: '공부', '음악', '재미' 등 원하는 카테고리를 직접 추가하고 관리할 수 있습니다.
- **비주얼 메모 리스트**: 저장된 메모 리스트의 배경으로 **유튜브 영상 썸네일**이 자동으로 표시되어 시각적으로 훨씬 찾기 편합니다.
- **프리미엄 다중 레이어 디자인**:
    - **가독성 오버레이**: 썸네일 위에 어두운 그라데이션 오버레이를 적용하여 텍스트가 명확하게 보입니다.
    - **고대비 타이포그래피**: 모든 텍스트에 화이트 컬러와 깊은 그림자(Shadow)를 적용하여 어떤 배경에서도 선명하게 읽힙니다.
    - **프로스트 글래스 효과**: 카테고리 태그에 모던한 반투명 유리(Blur) 효과를 적용했습니다.
- **Global Dark Mode**: 윈도우 다크 모드와 완벽하게 어우러지는 챠콜 톤의 프리미엄 다크 테마를 제공합니다.
- **유튜브 감성 accent**: 유튜브의 아이덴티티를 살린 생동감 넘치는 레드 포인트 컬러를 사용합니다.
- **정렬 및 필터링**:
    - 카테고리별로 모아보기
    - 최신순 / 오래된순 / 카테고리순 정렬 지원

## 🚀 설치 방법

이 확장 프로그램은 현재 크롬 웹 스토어에 등록되어 있지 않으며, **개발자 모드**를 통해 로컬에서 설치하여 사용할 수 있습니다.

### 1. 소스 코드 다운로드
- 이 저장소를 `git clone` 하거나 ZIP 파일로 다운로드하여 압축을 풉니다.
  ```bash
  git clone https://github.com/YOUR_USERNAME/YouTubeMemoExtension.git
  ```

### 2. 크롬 확장 프로그램 로드
1. Chrome 브라우저를 실행하고 주소창에 `chrome://extensions`를 입력합니다.
2. 우측 상단의 **개발자 모드 (Developer mode)** 스위치를 켭니다.
3. 좌측 상단의 **압축해제된 확장 프로그램을 로드합니다 (Load unpacked)** 버튼을 클릭합니다.
4. 다운로드 받은 폴더(`YouTubeMemoExtension`)를 선택합니다.

### 3. 사용하기
- 유튜브 영상 페이지로 이동합니다.
- 브라우저 우측 상단의 확장 프로그램 아이콘(퍼즐 조각 모양)을 클릭하여 **YouTube Memo**를 고정(핀)하면 더 편하게 쓸 수 있습니다.
- 아이콘을 클릭하여 메모를 작성하고 저장하세요!

## 🛠 기술 스택
- **Frontend**: HTML5, CSS3 (Vanilla CSS with backdrop-filters), Vanilla JavaScript
- **Storage**: Chrome Storage API (`chrome.storage.local`)
- **Manifest**: Manifest V3

```
YouTubeMemoExtension/
├── icons/            # 아이콘 이미지

├── popup.html        # 팝업 UI HTML
├── popup.js          # 팝업 로직 및 스토리지 관리 JS
├── style.css         # UI 스타일링 (프리미엄 다크 모드 및 썸네일 효과)
└── README.md         # 프로젝트 설명 문서
```
